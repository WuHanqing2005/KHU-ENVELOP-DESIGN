# 基于Photoshop的庆熙大学信封设计

### ✨ 项目概述

本项目开源了一个利用 **Adobe Photoshop (.psb)** 深度定制的 **韩国庆熙大学** 主题信封设计模板。此模板专为提高效率而设计，它完美结合了 Photoshop 强大的 **数据组（Data Sets）** 功能，实现了**收发件人信息的自动化批量填充与导出**。

作为一名在**庆熙大学电子工程系**的学生，我致力于将设计美学与实用技术相结合。这个模板不仅具有精美的庆熙大学元素，更提供了一个高效的信函批量处理解决方案。

## 🛠️ 核心功能：Photoshop 数据组批量导出

利用 Photoshop 的“数据组”功能，您可以将大量收发件人信息导入，并**一键生成**所有对应的信封图片，极大地简化了批量寄信的准备工作。

### 1\. 数据源与结构 (`datalist.csv`)

项目使用标准的 **CSV 文件**作为批量数据输入源。

  * **文件路径：** `datalist.csv`
  * **推荐编辑工具：** Excel、Google Sheets 或 VSCode。

| 字段名 | 描述 | 对应 Photoshop 图层名 | 备注 |
| :--- | :--- | :--- | :--- |
| **NAME\_FROM** | 发件人姓名 | `NAME_FROM` | 必须与图层名完全一致 |
| **TEL\_FROM** | 发件人电话 | `TEL_FROM` | |
| **ADDRESS\_FROM** | 发件人地址 | `ADDRESS_FROM` | |
| **NAME\_TO** | 收件人姓名 | `NAME_TO` | |
| **TEL\_TO** | 收件人电话 | `TEL_TO` | |
| **ADDRESS\_TO** | 收件人地址 | `ADDRESS_TO` | |

> **CSV数据集示例 (`datalist.csv`)：**
>
> 为保护用户隐私，以下示例信息使用占位符，实际使用时请替换为真实地址。
>
> ```csv
> NAME_FROM,TEL_FROM,ADDRESS_FROM,NAME_TO,TEL_TO,ADDRESS_TO
> 张三,+86-100-2000-3000,"XX省XX市XX路XX号",李四,+86-200-3000-4000,"YY市YY区YY街YY楼"
> 王五,+82-10-xxxx-xxxx,"대한민국 서울시",Kim Dami,+1-999-xxxx-xxxx,"USA, New York"
> ```

### 2\. 模板准备与字体设置

在 Photoshop 模板文件 `KYUNG HEE UNIVERSITY.psb` 中：

  * **图层命名：** 信封上所有可变文本区域（如地址、姓名）都对应一个独立的文本图层，且图层名称**精确对应**上述 CSV 字段名（例如：`ADDRESS_FROM` 对应 `ADDRESS_FROM` 变量）。
  * **字体兼容性调整：** 这是一个关键步骤。由于您的 CSV 数据可能包含**中文、韩文、英文**等多种语言，请在批量导出前：
    1.  打开 `KYUNG HEE UNIVERSITY.psb`。
    2.  **分别选中**包含多语种的图层（如 `ADDRESS_FROM` 或 `ADDRESS_TO`）。
    3.  在“字符”面板中，预先设置好能支持对应语种的最佳**字体**和**字号**。

### 3\. 批量导出操作流程

1.  **准备数据：** 确保 `datalist.csv` 文件已完成编辑并保存。
2.  **导入数据组：**
      * 在 Photoshop 中，导航至 **“图像” -\> “变量” -\> “数据组” (Image -\> Variables -\> Data Sets)**。
      * 点击 **“导入”**，选择 `datalist.csv` 文件，并加载。您可以在预览窗口中切换不同的数据组（即 CSV 中的每一行）进行检查。
3.  **批量导出文件：**
      * 导航至 **“文件” -\> “导出” -\> “数据组文件” (File -\> Export -\> Data Sets as Files)**。
      * 设置输出目录和文件命名规则。
      * 点击 **“确定”**，Photoshop 将自动为您 CSV 文件中的每一行数据生成一个完整的信封图片（例如 PNG 或 JPEG 格式）。

## 📐 信封打印与制作指南

导出的图片文件设计为适应 A4 纸张，方便家庭或办公室打印。

1.  **打印：** 将导出的信封文件以 **1:1 全尺寸** 打印在 A4 纸张上。
2.  **裁剪：**
      * 使用剪刀或美工刀，沿着图示中边缘画有**删除线的部分**进行精确裁剪。
3.  **折叠与粘贴：**
      * 沿着信封设计中的内部折痕线进行折叠。
      * 使用胶水或双面胶将边缘侧翼粘贴固定。
      * **最终成品尺寸：** 完成制作后，您将得到一个标准的信封，尺寸约为 **180 mm x 115 mm**。
